# Astera Labs — Hyperfocused Schema

## Investment Mandate

Astera Labs makes the connectivity chips that tie together the components inside AI servers and between servers — PCIe retimers, CXL controllers, and optical interconnects. Every high-end AI server (H100, Blackwell, Vera Rubin) contains Astera chips. The structural growth driver is CXL memory pooling: if CXL becomes the memory architecture for AI data centers, Astera is the connectivity fabric for that transition.

**Core thesis:** AI server BOM expansion benefits Astera on every unit. CXL adoption (2027+) is the second-wave structural opportunity that current revenue doesn't yet price. Atreides Management (Gavin Baker) holds $369M — their top position.

## Standing Knowledge

### Product Lines

- **Aries (PCIe/CXL retimers):** The core current revenue driver. Retimers extend the viable distance of PCIe signals inside a server, enabling larger, more complex server designs. Required in AI servers with multiple GPUs, CPUs, and memory modules. Ships in every NVIDIA H100, H200, and Blackwell server.
- **Taurus (Ethernet retimers):** For rack-to-rack ethernet connectivity inside AI clusters.
- **Leo (Optical interconnects — Smart Cable Modules):** Enables rack-scale optical connectivity. The Scorpio X product (announced at Amazon's TMC Conference May 2026) is Leo deployed for Amazon Trainium3 clusters at scale — and management confirmed Scorpio X becomes the largest product line by year-end 2026.

### CXL Memory Pooling — The 2027+ Thesis

CXL (Compute Express Link) is a protocol that allows CPUs, GPUs, and memory to share a high-bandwidth, low-latency memory pool across a rack or cluster. Instead of each GPU having dedicated HBM, memory is pooled and dynamically allocated. Astera's CXL controllers are the connectivity layer for this architecture.

Microsoft and Meta are running CXL pilots. The standard is still maturing (CXL 3.0). Commercial scale adoption is most likely 2027–2030. This is a real optionality layer on top of Astera's existing retimer revenue.
