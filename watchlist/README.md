# Watchlist — Next Frontier Candidates

Saturn scouts for new AI/semiconductor supply chain chokepoints beyond the current 7 positions. This directory holds candidate pages for companies that pass the thesis filter but haven't been added as full satellite positions yet.

## Thesis Filter

A company belongs on the watchlist if it answers YES to all three:
1. **Choke point:** Does it supply something the AI buildout cannot proceed without?
2. **Stack-agnostic:** Does it get paid regardless of which AI application or GPU architecture wins?
3. **Durable:** Is the position defensible for 5–10 years (IP, switching costs, monopoly, scale)?

## Candidate Template

Each candidate gets a file named `[TICKER].md` with:
- **Why scouted:** what source surfaced it and when
- **Choke point thesis:** what it controls in the supply chain
- **Risk:** why it might not make the cut
- **Status:** Watching / Researching / Promoted / Rejected
- **Promote trigger:** what would make this a full satellite position

## Graduation

A watchlist candidate becomes a full entity when:
- Choke point thesis is confirmed by 3+ independent sources
- Full schema and DCF inputs have been created
- Portfolio holder (Arvs) approves the addition
